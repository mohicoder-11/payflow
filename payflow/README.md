# PayFlow 💳

Premium fintech app for Mohindharan DK — built with React.

---

## 🚀 Deploy to Vercel (Free, 3 minutes)

### Step 1 — Create GitHub repo
1. Go to https://github.com/new
2. Name it `payflow`
3. Set to **Public**, click **Create repository**

### Step 2 — Upload your code
On the GitHub page for your new repo, click **"uploading an existing file"** and drag the entire `payflow/` folder contents in. Commit.

Or if you have Git installed:
```bash
cd payflow
git init
git add .
git commit -m "Initial PayFlow app"
git remote add origin https://github.com/YOUR_USERNAME/payflow.git
git push -u origin main
```

### Step 3 — Deploy on Vercel
1. Go to https://vercel.com and sign in with GitHub
2. Click **"Add New Project"**
3. Select your `payflow` repo
4. Vercel auto-detects React — just click **Deploy**
5. Done! Your app is live at `https://payflow-xxx.vercel.app`

### Step 4 — Add to iPhone Home Screen
1. Open your Vercel URL in **Safari** on iPhone
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Name it **PayFlow** → tap **Add**
5. It now works like a native app! 🎉
